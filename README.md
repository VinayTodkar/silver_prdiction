# 🪙 AI Silver Price Prediction Dashboard

An AI-powered silver price forecasting dashboard using:

- Streamlit
- LSTM Deep Learning
- Yahoo Finance API
- Candlestick Charts
- Moving Averages

---

# 🚀 Features

✅ Live Silver Market Data  
✅ LSTM AI Prediction  
✅ Candlestick Chart  
✅ KPI Dashboard  
✅ Future Price Prediction  
✅ CSV Download  
✅ Moving Average Analysis
